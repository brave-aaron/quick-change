# QuickChange - Landing Page de Conversion de Devises

Bienvenue sur le dépôt du projet QuickChange, développé dans le cadre du programme de stage Full Stack Web Development chez DecodeLabs (https://github.com/brave-aaron/quick-change.git).

QuickChange est une Landing Page moderne, minimaliste et entièrement responsive qui permet aux utilisateurs de simuler des conversions monétaires instantanées (EUR, USD, XOF) et de consulter les taux du jour.

---

## Objectifs du Jalon 1
L'objectif de cette première étape était de poser des bases solides en développement Frontend :
- Sémantique HTML5 : Utilisation de balises structurées (header, main, section, footer) pour un code propre et accessible.
- Design System et Charte Graphique : Intégration rigoureuse de la palette de couleurs imposée (Mocha Mousse #A5856F, Ethereal Blue #A0D4E0, et Moonlit Grey #F2F0EA).
- Responsive Design (Mobile-First) : Interface optimisée d'abord pour les smartphones (grille sur 1 colonne), s'adaptant de manière fluide aux écrans d'ordinateurs (grille sur 2 colonnes).
- Logique JavaScript Native (Vanille JS) : Dynamisation du formulaire avec un script local simulant le calcul de conversion et gérant la validation des saisies.

---

## Technologies Utilisées
- HTML5 : Structuration sémantique du contenu.
- CSS3 : Flexbox, CSS Grid pour la mise en page responsive, et scroll-behavior: smooth pour la navigation par ancres.
- JavaScript (ES6+) : Logique de calcul locale et manipulation du DOM.
